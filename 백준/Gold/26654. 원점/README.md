### 문제 설명

<p>희원이는 변이 $N$개이고 넓이가 $A$인 정다각형을 가지고 있다. 넓이가 $B$인 원이 위치한 평면에 가지고 있는 정다각형을 놓으려고 할 때, 원의 내부 또는 원주 위에 위치할 수 있는 정다각형 꼭짓점의 최대 개수를 구하여라.</p>

### 입력 

 <p>첫 번째 줄에 테스트 케이스의 개수 $T$가 주어진다. $(1 \leq T \leq 10\,000)$</p>

<p>다음 $T$개의 줄에 정다각형의 변의 개수 $N$과 정다각형과 원의 넓이 $A$, $B$가 공백으로 구분되어 정수로 주어진다. $(3 \leq N \leq 1\,000\,000;$ $1 \leq A, B \leq 10^9)$</p>

### 출력 

 <p>각 테스트 케이스에 대해, 원의 내부 또는 원주 위에 위치할 수 있는 정다각형 꼭짓점의 최대 개수를 한 줄에 하나씩 출력한다.</p>
